# Actividad 3.3

**Fecha de entrega:** 8 de mayo de 2026, 23:59

**Instrucciones:**

Realizar la práctica de **LinkServer** como se mostró en clase. Deberás generar un reporte detallado en donde se muestre el funcionamiento y la evidencia de cada configuración que se realice para que la práctica quede tal cual se mostró en clase.

---

### Resumen de pasos a realizar:

1. **Instalar una máquina virtual con Windows** y configurar la conexión de red para que la máquina host y la virtual tengan comunicación.
2. **Instalar una versión de SQL Server** en la máquina virtual.
3. **Configurar el acceso remoto** al servidor.
4. **Conectar mediante LinkServer** el SQL Server de la máquina virtual al SQL Server de la máquina host (principal).
5. **Instalar MySQL** en la máquina host.
6. **Configurar la conexión de MySQL al SQL Server** de la máquina host mediante LinkServer.
7. **Realizar pruebas de conectividad:** Una vez conectados los servidores al SQL Server del host, realizar pruebas de crear una tabla, insertar datos, borrar datos, etc., desde el SQL Server Host y también desde el SQL Server de la máquina virtual y del servidor MySQL.
8. **Migración de datos:** Pasar desde el SQL Server de la máquina host al SQL Server de la máquina virtual la base de datos de la práctica de los archivos de Excel; igualmente, desde el SQL Server host pasar la misma base de datos pero ahora al MySQL.
9. **Validación de datos:** Realizar las consultas de conteo que se pidieron en la práctica de los archivos de Excel en el SQL Server del Host, en el SQL Server de la máquina virtual y en el MySQL para evidenciar que la base de datos en cada servidor tiene las mismas tablas y el mismo volumen de registros en cada tabla.
