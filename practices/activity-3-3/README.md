# Actividad 3.3

Estructura base para trabajar la actividad 3.3 desde el inicio.

## Archivos

- `instructions.md`: Instrucciones oficiales de la actividad.
- `queries.sql`: Script principal de desarrollo.
- `REPORTE.md`: Reporte de evidencias y justificación.
- `rename-img.sh`: Utilidad para renombrar capturas como `execN.png`.
- `assets/img/`: Carpeta para capturas.

## Flujo sugerido

1. Redactar o pegar el desarrollo en `queries.sql`.
2. Ejecutar por bloques y capturar evidencias en `assets/img`.
3. Renombrar captura nueva con `./rename-img.sh`.
4. Documentar cada bloque en `REPORTE.md`.
