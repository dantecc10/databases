# Actividad 1

Responder las siguientes preguntas:

1. ¿Qué es una base de datos?

Una base de datos es un repositorio digital para almacenar, gestionar y proteger colecciones organizadas de datos. Abarca desde el hardware físico en el que se almacenan los datos, hasta el software que organiza y controla el acceso a estos.

2. ¿Qué tipos de bases de datos existen?

Las organizaciones utilizan diferentes tipos de bases de datos para gestionar diferentes tipos de datos y admitir diferentes aplicaciones. Algunos de los más comunes son:
- Bases de datos de navegación: Almacenan datos en conjuntos de registros vinculados. Las hay jerárquicas y de red; las primeras organizan los datos en una estructura similar a un árbol de registros principales y secundarios, y se navega de arriba hacia abajo; las segundas permiten que un registro secundario apunte a varios principales y permite navegaciones más flexibles.
- Bases de datos relacionales: almacenan datos en tablas, que contienen información sobre un tipo de entidad.
- Bases de datos no relacionales o bases de datos NoSQL: Surgieron para admitir tipos de datos no estructurados y semiestructurados, como texto e imágenes, que no encajan en las relacionales. Las hay
 * Gráficas
 * De documentos
 * De datos clave-valor
 * De columna ancha
- Bases de datos orientadas a objetos: Llevan el paradigam de la POO (*Programación Orientada a Objetos*) como opción de almacenamiento de datos.
- Bases de datos vectoriales: Almacenan información como matrices de números llamados vectores, que se agrupan en función de la similitud.
- Bases de datos en la nube: Son alojadas en la nube.
